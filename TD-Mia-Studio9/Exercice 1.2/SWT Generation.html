Exercice 1.2

Rajouter le code minimal d'initialisation du shell

  • Mettre en place la generation du constructeur tel qu'indiquée ci-dessous
  • Mettre en place la generation de la methode createContents, cette methode doit exploiter le champ description de la vue courante comme titre de la fenêtre.
  • Tester avec la runconfig

Exemple :
package com.mia.formation.generation.ui;

import org.eclipse.swt.widgets.Shell;
import org.eclipse.swt.widgets.Button;

public class ClientView extends Shell {

	private Button btnQuitter;
	private Button btnAfficherCompte;

	/**
	 * Create the shell.
	 * @param display
	 */
	public ClientView(Display display) {
		super(display, SWT.SHELL_TRIM);
		createContents();
	}

	/**
	 * Create contents of the shell.
	 */
	protected void createContents() {
		setText("ClientView");
		setSize(450, 300);

	}


}
