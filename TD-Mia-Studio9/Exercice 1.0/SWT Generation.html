Exercice 1.0

Generer une classe Java héritant de org.eclipse.swt.widgets.Shell pour chaque instance de View du modèle

  • Créer un File Template attaché à View
  • Générer la déclaration Java de la classe, en choisissant un package Java donné (ex : "com.mia.formation.generation.ui" )
  • Générer un nom de fichier correspondant de la forme "xxx/name.java" (ex : com/mia/formation/generation/ui/ClientView.java
  • Creer un Scenario attaché à View qui appelle le File Template
  • Tester à travers la création d'un runconfig

Exemple de fichier généré pour une intance de View:
package com.mia.formation.generation.ui;

import org.eclipse.swt.widgets.Shell;

public class ClientView extends Shell {
}

Avec comme nom de fichier :
com/mia/formation/generation/ui/ClientView.java
