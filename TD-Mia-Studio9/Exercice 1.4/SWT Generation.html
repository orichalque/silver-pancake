Exercice 1.4

(Optionnel) Tester l'appli avec le code minimal de lancement

  • Inserer dans une classe utilitaire Java le code ci-dessous pour instancier et tester la classe BanqueView generee

Exemple :
	public static void main(String[] args) {
		Display display = Display.getDefault();
		
		
		BanqueView banqueView = new BanqueView(display);
		banqueView.open();
		banqueView.layout();
		
		while (!banqueView.isDisposed()) {
			if (!display.readAndDispatch()) {
				display.sleep();
			}
		}

	}
