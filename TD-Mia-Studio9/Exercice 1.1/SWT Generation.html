Exercice 1.1

Generer des variables d'instance pour les boutons des actions

  • Créer un template sur la métaclasse Action qui génère la déclaration d'une variable d'instance de Button
  • Mettre en place, depuis le file template sur View, l'appel en boucle des generations de boutons pour les Action> liées à la View courante.
  • Mettre en place l'import de org.eclipse.swt.widgets.Button
  • Tester depuis la runconfig.

Note sur l'appel en boucle :
on utilisera de préférence la notation pointée du langage de template.
Variante 1.1.b : Dans un second temps, on utilisera une macro Java, et l'api apply

Exemple de fichier généré :
package com.mia.formation.generation.ui;

import org.eclipse.swt.widgets.Shell;
import org.eclipse.swt.widgets.Button;

public class ClientView extends Shell {

	private Button btnQuitter;
	private Button btnAfficherCompte;

}
